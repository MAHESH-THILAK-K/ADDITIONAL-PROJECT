## 📌 Overview

A **compact and efficient USB to UART converter** built using the **CP2102 IC**, enabling reliable serial communication between your PC and embedded systems.

Designed for **firmware flashing, debugging, and real-time data exchange** with platforms like ESP32, Arduino, and more.

## ⚡ Key Features

* ⚡ CP2102 USB-to-UART bridge
* 🔌 USB Type-C interface
* 🔁 Stable & high-speed communication
* 📡 Dedicated TX / RX lines
* ⚙️ Dual voltage support (5V / 3.3V)
* 🔧 Plug & Play operation
* 📏 Compact & clean PCB layout

---

## 🔌 Pin Configuration

| Pin  | Description |
| ---- | ----------- |
| GND  | Ground      |
| VBUS | 5V Output   |
| RX   | Receive     |
| TX   | Transmit    |
| RST  | Reset       |

## 🧰 Hardware Breakdown

| Component          | Description            |
| ------------------ | ---------------------- |
| CP2102             | USB to UART Bridge IC  |
| USB-C Port         | Power + Data Interface |
| Header Pins        | Serial Connectivity    |
| Passive Components | Filtering & Stability  |

## 🚀 Getting Started

```bash
1. Connect USB-C to PC
2. Cross connect TX ↔ RX
3. Power the target board
4. Upload firmware / open serial monitor
```
